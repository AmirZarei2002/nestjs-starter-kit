# ESLint Import Rules - Quick Reference

## 🚀 Quick Fixes

### ❌ Error: `src/` import detected

```typescript
// Before
import { CreateCategoryDto } from 'src/modules/category/application/dtos/create-category.dto';

// After
import { CreateCategoryDto } from '@category/application';
```

### ❌ Error: Infrastructure import detected

```typescript
// Before
import { UserRepository } from '@user/infrastructure/prisma/persistence/user.repository';

// After (use domain interfaces)
import { IUserRepository } from '@user/domain';
```

### ❌ Error: Import order

```typescript
// Before (wrong order)
import { CategoryEntity } from '@category/domain';
import { Injectable } from '@nestjs/common';
import { PrismaService } from '@prisma/prisma.service';

// After (correct order)
import { Injectable } from '@nestjs/common';
import { PrismaService } from '@prisma/prisma.service';
import { CategoryEntity } from '@category/domain';
```

### ❌ Error: Empty lines between imports

```typescript
// Before
import { Module } from '@nestjs/common';

import { PrismaService } from '@prisma/prisma.service';

// After
import { Module } from '@nestjs/common';
import { PrismaService } from '@prisma/prisma.service';
```

## 📋 Import Order Starter-kit

Copy this order for your files:

```typescript
// 1. NestJS packages
import { Injectable, Module } from '@nestjs/common';

// 2. External packages
import { SomePackage } from 'external-package';

// 3. Common utilities
import { PaginationQueryDto } from '@common/dtos/pagination-query.dto';

// 4. Shared services
import { CustomLoggerService } from '@shared/services/custom-logger.service';

// 5. Config
import { SomeConfig } from '@config/some.config';

// 6. Prisma
import { PrismaService } from '@prisma/prisma.service';

// 7. Module-specific (from other modules)
import { UserEntity } from '@user/domain';

// 8. Current module (barrel exports)
import { CategoryEntity } from '@category/domain';

// 9. Parent directory (relative)
import { BaseEntity } from '../base.entity';

// 10. Same directory (relative)
import { CreateCategoryDto } from './create-category.dto';
```

## 🎯 Available Path Aliases

```typescript
@common/*       → src/common/*
@shared/*       → src/shared/*
@config/*       → src/config/*
@prisma/*       → src/prisma/*
@modules/*      → src/modules/*
@category/*     → src/modules/category/*
@user/*         → src/modules/user/*        (add as needed)
@product/*      → src/modules/product/*     (add as needed)
```

## 🛠️ Commands

```bash
# Check for errors
npm run lint

# Auto-fix (fixes order, formatting)
npm run lint -- --fix

# Check specific file
npx eslint src/modules/category/**/*.ts
```
